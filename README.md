RetailIQ is a retail analytics and sales prediction project built using Python and Streamlit. The project analyzes retail sales data, generates business insights through interactive visualizations, and predicts future sales trends using machine learning techniques.

The dashboard helps businesses understand customer purchasing behavior, product performance, outlet efficiency, and overall sales distribution to support data-driven decision-making.

Features Interactive sales analytics dashboard Data preprocessing and cleaning Exploratory Data Analysis (EDA) Sales and profit visualization using Plotly Machine learning-based sales prediction Dynamic filtering and business insights User-friendly Streamlit interface Technologies Used Python Pandas NumPy Matplotlib Plotly Scikit-learn Streamlit Project Structure RetailIQ/ │ ├── data/ │ └── retail_dataset.csv │ ├── models/ │ └── trained_model.pkl │ ├── app.py ├── requirements.txt ├── README.md └── notebooks/ └── analysis.ipynb Installation Clone the repository git clone https://github.com/your-username/retailiq.git Navigate to the project directory cd retailiq Install dependencies pip install -r requirements.txt Running the Project

Start the Streamlit application using:

streamlit run app.py
